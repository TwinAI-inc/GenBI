Data Analysis & Insights
💬
Chart Assistant
/api/chart-assist
Conversational chart builder — describe what you want in plain English (e.g. "Show sales by region as a bar chart") and the AI generates the chart configuration with type, axes, aggregation, and colors.
Chart Assistant button in Analytics header
🔍
Ask Data
/api/ask-data
Free-form natural language Q&A over your dataset. Ask questions like "What's the average revenue?" or "Which region has the highest margin?" and get specific numerical answers with explanations.
"Ask anything about your data..." input on Analytics
💡
Auto Insights
/api/auto-insights
Automatically analyzes the uploaded dataset and surfaces notable patterns, trends, and observations. Displayed as an AI Insights briefing card on the Analytics page.
Loads automatically when data is present
⚠️
Anomaly Detection
/api/anomaly-detect
Identifies unusual patterns, outliers, and data quality issues. Displays severity-coded badges (high / medium / low) directly on affected charts.
Runs automatically after chart render
Chart Intelligence
📝
Chart Narratives
/api/chart-narrative
Generates a plain-English one-sentence caption for each chart, explaining what it shows and the key business insight — displayed below chart titles for instant context.
Runs automatically after chart render
📈
Forecast
/api/forecast
Predicts future values for time-series line charts. Extends the chart with AI-generated forecast data for the next 3 periods using the last 20 data points.
"Forecast" button on line charts
🧭
Suggested Actions
/api/suggest-actions
Context-aware recommendations for what to do next — adapts based on current app state (no data → suggests upload; data loaded → suggests specific analyses).
Loads automatically on Analytics page
🎯
Recommendations
/api/recommendations
Generates actionable business recommendations based on identified key influencers. Suggests specific actions to improve the target metric, with impact assessments.
"Recommendations" button on Key Influencers
Data Quality & Understanding
🛡️
Data Quality Analysis
/api/data-quality
Comprehensive dataset assessment including missing values, data type consistency, outlier detection, and quality metrics. Returns findings with severity levels and improvement recommendations.
"Analyze Quality" button on Documents page
📋
Column Descriptions
/api/describe-columns
Auto-generates a data dictionary by analyzing column names, data types, and sample values — produces meaningful descriptions for every field in the dataset.
"Generate" button on Documents page
🔑
Key Influencers
/api/key-influencers
Statistical analysis identifying which categorical factors most strongly influence a selected metric. Calculates influence multipliers and ranks both positive and negative drivers.
"Analyze" button on Key Influencers page
❓
Explain Influencer
/api/explain-influencer
Provides a data-driven explanation for why a specific factor was identified as a key influencer — shows the causal relationship between the factor value and metric behavior.
"Why?" button next to selected influencer
